## Tips for Recording

1. **Screen Recording:**
   - Use OBS Studio, Loom, or QuickTime (Mac)
   - Record in 1080p if possible
   - Show clear, readable text

2. **Audio:**
   - Use a good microphone
   - Speak clearly and at moderate pace
   - Minimize background noise

3. **Preparation:**
   - Test all features before recording
   - Have sample data ready
   - Practice the script once

4. **Editing:**
   - Add text overlays for key points
   - Cut out long pauses
   - Add transitions between sections

5. **Upload:**
   - Upload to YouTube (unlisted) or Google Drive
   - Make sure link is accessible
   - Test the link before submitting
