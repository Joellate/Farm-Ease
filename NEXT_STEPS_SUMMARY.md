## Troubleshooting

**Can't deploy?**

- Render/Vercel have good documentation
- Free tiers are available

**Video too long?**
- Focus on core features (5-7 minutes is fine)
- Cut out pauses
- Show key functionality

**Google Doc not accessible?**
- Make sure sharing is set to "Anyone with link"
- Test the link in incognito mode
- Double-check all links work
